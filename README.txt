Lien du jeu de données avec Training.csv et Testing.csv
À glisser dans le répertoire data
https://www.kaggle.com/datasets/kaushil268/disease-prediction-using-machine-learning

Le projet contient :
    - un fichier nettoyage.py qui prépare le dataset au Machine Learning. Il doit être éxécuté en premier.
    - un fichier correlations.py qui calcule la matrice de corrélation pour savoir si nos colonnes sont corrélées à la colonne cible.
    - un fichier analyse_RandomForest.py qui utilise un algorithme de RandomForest pour créer un modèle permettant de prédire Testing.csv.
    - le repertoire .venv pour ouvrir le projet sous PyCharm
